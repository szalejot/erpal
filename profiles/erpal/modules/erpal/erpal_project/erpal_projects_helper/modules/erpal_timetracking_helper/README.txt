
ERPAL timetracking helper

----------------------

This module provides Timetracking views filter "Show subtasks".
Filter gets timetrackings of subtasks.
It provides simple checkbox "Show subtasks". 
You can expose this filter if it's necessary.

----------------------
Usage:
----------------------
1. Install module;
2. Create view for timetrackings node(or edit current);
3. Click add 'Filter criteria';
4. Choose group "Timetracking" at Filter select;
5. Choose "Timetracking: Show subtasks" filter;
6. Than you can expose this filter if it's necessary.
